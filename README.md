# 🏝️ Maui Book Club: La Familia Edition

Welcome to the official repository for **La Familia's Maui Book Club 2026**. This project is a custom-designed, tropical-themed web application built to coordinate our family's reading journey, track progress, and share spoiler-free predictions.

## 📖 Current Selection
**Title:** *The Guest List*  
**Author:** Lucy Foley  
**Theme:** Mystery / Island Thriller  

---

## ✨ Features

-   **Home Dashboard**: The central hub featuring our family beach photo and quick access to all club sections.
-   **Reading Progress**: Real-time tracking of everyone's page count with "nudge" notifications for those who need a boost.
-   **Spoiler-Free Predictions**: A dedicated space to log theories and suspects for our current mystery read, hidden from others until checkpoints are reached.
-   **Member Profiles**: "Our Literary Circle" showcasing reading stats and achievements for every family member.
-   **Mobile-First Design**: Optimized for seamless use on smartphones while we're on the go or lounging by the beach.

---

## 📂 Project Structure

This repository contains the full HTML and CSS for the following core screens:

1.  `index.html` (Home Dashboard)
2.  `book-details.html` (Reading Schedule & Checkpoints)
3.  `predictions.html` (Mystery Tracker)
4.  `progress.html` (Family Leaderboard)
5.  `profiles.html` (Member Statistics)
6.  `update-progress.html` (Personal Check-in Screen)
7.  `invite.html` (Family Onboarding Portal)

## 🚀 Getting Started

### Hosting with GitHub Pages
1.  **Upload**: Add all exported HTML and CSS files to this repository.
2.  **Enable Pages**: Go to **Settings > Pages**.
3.  **Deploy**: Select the `main` branch and `/root` folder, then click **Save**.
4.  **Share**: Your app will be live at `https://[your-username].github.io/[repo-name]/`.

### Local Preview
Simply download the files and open `index.html` in any web browser to view the app locally.
